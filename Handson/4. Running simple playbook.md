1. Create simple playbook

```
---
- hosts: all
  tasks:
  - name: Print Hello World
    debug: msg="Hello World"
```

2. Run the playbook

```
vagrant@ubuntu1804:~/ansible$ ansible-playbook hello.yml

PLAY [all] ******************************************************************************************************************************************************************************************************

TASK [Gathering Facts] ******************************************************************************************************************************************************************************************
ok: [192.168.7.104]

TASK [Print Hello World] ****************************************************************************************************************************************************************************************
ok: [192.168.7.104] => {
    "msg": "Hello World"
}

PLAY RECAP ******************************************************************************************************************************************************************************************************
192.168.7.104              : ok=2    changed=0    unreachable=0    failed=0
```
